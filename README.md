 ## ‚öôÔ∏è Game Mechanics
 
 ### Flight Tracking
 - The game automatically tracks all active planes in your simulation
 - Each plane generates passive income over time
 - More planes = more income and faster XP gain
 - Maximum planes depend on available flight data
 
 ### Time System
 - The game runs in real-time when not paused
 - Speed multipliers (2x, 3x, 5x) affect:
   - XP gain rate
   - Token earning rate
   - Passive income intervals
 
 ### Day/Night Cycle
 - The 3D globe features realistic day/night visualization
 - Brightness can be adjusted in settings
 - No gameplay impact, purely visual

 ## üí∞ Economy System
 
 ### SKYE Token
 The primary in-game currency used for:
 - Purchasing planes
 - Buying upgrades
 - Acquiring power-ups
 - Speed boosts
 
 ### Earning Tokens
 
 #### 1. **Passive Income** (every 30 seconds)
 - Base: `planeCount / 10`
 - Modified by: Plane Efficiency upgrade
 - Modified by: Passive Income upgrade
 - Lucky Charm: 25% chance to double
 
 #### 2. **Periodic Rewards** (every 60 seconds)
 - Base reward: 50 SKYE
 - Plane bonus: `(planeCount / 5) * 5`
 - Modified by: Plane Efficiency upgrade
 - Lucky Charm: 25% chance to double
 
 #### 3. **Mission Rewards**
 - Completing missions grants 50-200 SKYE
 - One-time rewards per mission
 
 #### 4. **Daily Rewards**
 - 500 SKYE (once per day)
 
 #### 5. **Login Bonuses**
 - First login: 100 SKYE
 - Account creation: 500 SKYE
 
 #### 6. **Selling Planes**
 - Sell 5 planes: +25 SKYE
 
 ### Spending Tokens
 - Add 5 planes: 50 SKYE
 - Speed boosts: 100-500 SKYE
 - Upgrades: 100-200 SKYE (scales with level)
 - Shop items: 75-300 SKYE

 ## üìà Progression System
 
 ### Experience Points (XP)
 - Earned continuously while playing
 - Base rate: 5 XP every 10 seconds
 - Modified by: XP Boost upgrade
 - Modified by: Speed multiplier
 - Bonus XP from:
   - Adding planes: +15 XP
   - Selling planes: +5 XP
   - Completing missions
 
 ### Leveling Up
 - Each level requires more XP (increases by 1.5x)
 - Level 1: 100 XP
 - Level 2: 150 XP
 - Level 3: 225 XP
 - And so on...
 
 **Rewards per level:**
 - `100 * current_level` SKYE tokens
 - Visual level-up animation
 - Unlock new features (coming soon)

 ## üí° Tips & Strategies
 
 ### Beginner Tips
 1. **Claim Daily Reward First**
    - Always claim your 500 SKYE daily pack
    - Available 3 seconds after game loads
 
 2. **Don't Rush Planes**
    - Start with the default 50 planes
    - Only add more when you have spare SKYE
    - Each plane costs 10 SKYE (50 SKYE for 5 planes)
 
 3. **Complete Missions**
    - Easy source of bonus SKYE
    - Just play naturally to complete most missions
 
 ### Intermediate Strategies
 1. **Invest in Upgrades Early**
    - Permanent benefits compound over time
    - Plane Efficiency gives best ROI early
 
 2. **Use Speed Boosts Wisely**
    - Don't waste speed boosts when AFK
    - Best used during active play sessions
    - 5x speed is expensive but very efficient
 
 3. **Lucky Charm Timing**
    - Use during long play sessions (8 hours)
    - Combine with speed boosts for maximum effect
 
 ### Advanced Optimization
 1. **Upgrade Balance**
    - Keep all upgrades within 2-3 levels of each other
    - Balanced upgrades > maxing one upgrade
 
 2. **Plane Management**
    - Optimal plane count: 100-150 for mid-game
    - More planes = higher maintenance cost
    - Selling planes gives instant SKYE for emergencies
 
 3. **Token Banking**
    - Keep 500+ SKYE in reserve
    - Enables flexibility for opportunities
    - Can afford any upgrade or boost when needed
 
 ### Idle Play Strategy
 1. Set speed to 1x (no cost)
 2. Maximize passive income upgrades
 3. Add maximum planes your SKYE allows
 4. Activate Lucky Charm before going AFK
 5. Leave browser tab open
 
 ### Active Play Strategy
 1. Use 3x or 5x speed boosts
 2. Focus on XP Boost upgrades
 3. Complete missions actively
 4. Monitor rewards and claim immediately

 ## ‚ùì FAQ
 
 **Q: How do I save my progress?**
 A: Currently, progress is saved in browser localStorage. Web3 integration coming soon.
 
 **Q: What happens if I close the browser?**
 A: Your game state is saved. You won't earn tokens while offline.
 
 **Q: Can I play on mobile?**
 A: Yes, but desktop is recommended for best experience.
 
 **Q: Are the flight paths real?**
 A: Yes, the game uses authentic flight data visualization.
 
 **Q: How often can I claim daily rewards?**
 A: Once every 24 hours (resets at midnight).
 
 **Q: What's the maximum level?**
 A: Currently no level cap, but XP requirements scale infinitely.
 
 **Q: Can I transfer SKYE tokens to other players?**
 A: Not yet. Blockchain integration will enable this.
